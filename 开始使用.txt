========================================
  XrayR 配置生成器 - 开始使用
========================================

🎉 恭喜！项目已经创建完成！

📁 项目位置：
   当前目录

🚀 快速开始（3步）：

   第一步：安装依赖
   ----------------
   Windows 用户：双击运行 install.bat
   Linux/Mac：在终端运行 npm install && cd client && npm install

   第二步：启动服务
   ----------------
   Windows 用户：双击运行 start.bat
   Linux/Mac：在终端运行 npm run dev

   第三步：访问系统
   ----------------
   打开浏览器访问：http://localhost:5173
   默认密码：admin123456

📚 重要文档：

   ⭐ QUICKSTART.md      - 5分钟快速上手指南
   ⭐ README.md          - 完整项目说明
   ⭐ USAGE.md           - 详细使用教程
   ⭐ config-example.yml - 配置文件示例
   
   📖 PROJECT_OVERVIEW.md - 项目架构说明
   📖 FILES.md            - 文件清单

🔧 配置文件：

   .env - 环境变量配置（重要！）
   
   必须修改的配置：
   - ADMIN_PASSWORD：管理员密码
   - JWT_SECRET：JWT密钥

⚠️ 安全提示：

   1. 首次使用后立即修改 .env 中的密码
   2. 不要将 .env 文件提交到版本控制
   3. 定期备份 data 目录

🎯 功能特性：

   ✅ 密码鉴权 + JWT Token 认证
   ✅ 多面板管理
   ✅ 多节点配置
   ✅ 支持 Vless/Vmess/Shadowsocks/Trojan
   ✅ 一键生成配置
   ✅ 一键复制配置

📞 获取帮助：

   遇到问题？
   1. 运行 test-setup.bat 检查环境
   2. 查看 USAGE.md 使用教程
   3. 查看 README.md 常见问题

🌟 使用流程：

   1. 登录系统
   2. 添加面板（配置面板域名和密钥）
   3. 选择面板
   4. 添加节点（配置节点信息）
   5. 查看配置
   6. 复制配置
   7. 部署到服务器

💡 提示：

   - 节点ID必须与面板中的节点ID一致
   - 证书文件路径必须正确
   - 建议先阅读 QUICKSTART.md

========================================

现在就开始使用吧！

1. 双击运行 install.bat（首次使用）
2. 双击运行 start.bat（启动服务）
3. 打开浏览器访问 http://localhost:5173
